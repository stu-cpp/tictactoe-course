# Штрафное задание №1. OstreamWriter

В файле [src/player/my_observer.hpp](src/player/my_observer.hpp) реализован класс `ConsoleWriter`, который печатает в консоль информацию об игре.
Статический метод `print_game_state` печатает состояние доски и используется, например, в [tests/test_my_player.cpp](tests/test_my_player.cpp).

Напишите аналогичный класс `OstreamWriter` так, чтобы он мог выводить информацию не только в консоль (`std::cout`), но и в любой объект класса `ostream`.
Метод печати доски не должен быть статическим.

Напишите для него два конструктора:
- конструктор по умолчанию (в этом случае печать происходит в консоль)
- конструктор, принимающий `std::ostream`

Используйте этот класс в `test_my_player.cpp`, записывая ход игры не только в консоль, но и в файл вида `test_my_player_vs_human_<дата>_<время начала>_log.txt`
